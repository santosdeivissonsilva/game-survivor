# Monsters Suvivor

A [libGDX](https://libgdx.com/) project generated with [gdx-liftoff](https://github.com/libgdx/gdx-liftoff).

This is a project created for studies and improvement skills of game creation with libGDX.

## Controls

- W / A / S / D - Move player
- R - Restart (when dead)
- SPACE - Start game (from controls screen)
